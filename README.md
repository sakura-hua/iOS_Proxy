# Remote Resources

本仓库收集并整理多个 Quantumult X / Shadowrocket 远程资源（snippet/conf/sgmodel），用于快速在 代理工具 中拉取和使用。

## 使用方法
在 Quantumult X → Remote → 添加 Remote，类型选择 **Rewrite**，填入以下 raw URL：

https://raw.githubusercontent.com/users/repository/main/xxx.snippet
